# 팀 랜덤 배정 시스템

Socket.IO 기반의 실시간 팀 랜덤 배정 웹 애플리케이션입니다.

## 주요 기능

### 1. 호스트 기능
- 세션 생성 및 관리
- 플레이어 목록 설정
- 팀 구성 설정 (팀 이름, 인원 수)
- 같은 팀 금지 제약 조건 설정
- 팀 랜덤 배정 실행
- 결과 초기화

### 2. 관전자 기능
- 세션 ID로 참가
- 실시간으로 설정 확인
- 배정 결과 확인

### 3. 팀 배정 알고리즘
- 최대 10,000번 시도로 제약 조건을 만족하는 조합 탐색
- 플레이어 수와 팀 인원 수 자동 검증
- 같은 팀 금지 제약 조건 자동 검증

## 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. 서버 실행
```bash
# 일반 실행
npm start

# 개발 모드 (자동 재시작)
npm run dev
```

### 3. 브라우저 접속
```
http://localhost:3000
```

## 사용 방법

### 호스트
1. "세션 생성" 버튼 클릭
2. 생성된 세션 ID 확인
3. 플레이어 이름 입력 (한 줄에 하나씩)
4. 팀 구성 설정 (팀 이름, 인원 수)
5. 필요시 같은 팀 금지 조건 설정
6. "팀 배정하기" 버튼 클릭

### 관전자
1. "세션 참가" 버튼 클릭
2. 호스트가 공유한 세션 ID 입력
3. 실시간으로 설정 및 결과 확인

## 기술 스택

- **백엔드**: Node.js, Express, Socket.IO
- **프론트엔드**: Vanilla JavaScript, HTML5, CSS3
- **실시간 통신**: Socket.IO

## 주의사항

- 세션은 1시간 후 자동으로 삭제됩니다
- 세션 ID는 6자리 영문 대문자입니다
- 플레이어 수와 팀 인원 수의 합이 일치해야 합니다
